## 1. 项目概述

### 1.1 项目简介

SynPharm是一个AI驱动的智能药物研发平台，包含三个主要部分：
- **PC端Web界面**：基于Vue 3的药物发现平台Web界面
- **Java后端服务**：Spring Boot微服务架构，处理业务逻辑
- **Python AI服务**：FastAPI + PyTorch，负责核心预测算法

平台专注于药物靶点相互作用预测、分子可视化和药物研发数据管理。

### 1.2 核心功能

| 功能模块 | 描述 |
|---------|------|
| PPI预测 | 蛋白质-蛋白质相互作用预测 |
| DTI预测 | 药物-靶点相互作用预测 |
| DDI预测 | 药物-药物相互作用预测 |
| 3D可视化 | 分子结构交互式3D展示 |
| 任务管理 | 预测任务创建、追踪与管理 |
| 结果管理 | 预测结果查看、筛选与导出 |
| 个人中心 | 用户信息管理与偏好设置 |

## 18. 后端技术架构

### 18.1 架构设计

后端采用分层单体架构，包含以下层级：

1. **Controller层**：处理HTTP请求，参数校验，调用Service
2. **Service层**：业务逻辑处理，事务管理
3. **Repository层**：数据访问，与数据库交互
4. **Entity层**：数据库实体映射
5. **DTO层**：数据传输对象（请求/响应）
6. **Config层**：配置类（安全、跨域、MyBatis等）
7. **Utils层**：工具类（JWT、预测算法）
8. **Exception层**：异常处理（业务异常、全局异常）

### 18.2 核心特性

- **分层架构**：各层职责清晰，便于维护和扩展
- **统一响应**：所有接口返回相同格式
- **异常处理**：全局异常捕获，业务异常与系统异常分离
- **JWT认证**：无状态认证，支持Token过期验证
- **自动填充**：创建/更新时间自动填充
